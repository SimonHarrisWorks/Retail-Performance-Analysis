# Retail Performance Analysis
A retail analytics project exploring throughput, order size behaviour, revenue trends, and short‑term forecasting using Python.


## Project Overview
This project analyses retail transaction data to understand customer behaviour and operational performance.
It focuses on throughput patterns, order size dynamics, rolling revenue trends, and a short‑term revenue forecast to support decision‑making and identify operational cycles.


## Objectives
- Analyse daily throughput (orders per day)

- Measure average order size

- Build rolling‑window revenue trends

- Create a short‑term revenue forecast

- Identify operational cycles and anomalies


## Key Features
- Data cleaning and invoice aggregation

- Throughput and order size analysis

- Rolling revenue modelling

- Linear forecasting

- Clear visualisations

- Reproducible notebook workflow


## Tools & Technologies
- Python

- Pandas

- NumPy

- Matplotlib / Seaborn

- Jupyter Notebook


## How to Use
- Clone or download the repository

- Open the Jupyter notebook

- Run all cells to reproduce the analysis

- View generated graphs and forecast output
